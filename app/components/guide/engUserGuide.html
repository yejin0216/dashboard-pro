    Introduce

    Dashboard Pro2 is a UI(User Interface) service for visualizing IoT data such as devices, sensors, and events.
    Customize your dashboards and widgets to create your own IoT service UI.

    Features

    • 1) Real-time monitoring and control
      • You can collect and control various resource(device, sensor, event, etc.) information of IoT device in real time.
    • 2) Configuring User-customized UI
      • Widgets can be deployed to configure the UI for a variety of purposes.
    • 3) Free widget resizing and placement
      • You can change the size of the created widget to be larger or smaller, and change the position of the widget freely.
    • 4) Responsive web design
      • Dashboard Pro2 can be accessed through a browser on a variety of devices, including PCs, Tablets, and Smart Phones.

    Main Menu

    Bookmark

    A collection of shortcuts to frequently used or important devices. 2)Click on the area to move to the selected dashboard.

    • ◎ Description
      • 1) Add to Bookmarks: You can add a dashboard to your bookmarks list by selecting it and clicking the Add bookmarks button.
      • 2) Bookmark Name: This is the area where the name of the dashboard added to bookmarks is displayed. Click to go to that dashboard.
      • 3) Bookmark Information: You can see the details of the dashboard such as dashboard breakdown, worker(creator), and latest modification date.

    My devices

    You can inquire device and sensor information registered in IoTMakers, and define sensor attributes such as Group tag and Capability.

    • ◎ Description
      • 1) Device List: List of my devices registered in IoTMakers.
      • 2) Device information: Device name, model, and brief information of device can be checked.
      • 3) Details: Click the [Details] button to check device details.
      • 4) Sensor View: Click the [View Sensor] button to check sensor information that is dependent on the device.
      • 5) Capability Setting: Click the [Capability Setting] button to set the UI type of the control sensor that is dependent on the device model. The set UI type is applied to "Device image widget".

    Dashboard(list)

    You can register, edit, or delete new dashboards.

    • ◎ Description
      • 1) Add Dashboard: You can create a new dashboard by clicking the Add Dashboard button.
      • 2) Dashboard list: A list of dashboards you have created.
      • 3) Dashboard Information: Details of the dashboard, including dashboard names, dashboard breaks, etc.
      • 4) Details: Go to the selected dashboard screen.
      • 5) Edit: allows you to edit the name of the selected dashboard, etc.
      • 6) Delete: You can delete the selected dashboard.

      • 1) Refresh: You can see the new log by clicking the
      • 2) Informational message: Up to 100 events can be displayed per event from today's events.
