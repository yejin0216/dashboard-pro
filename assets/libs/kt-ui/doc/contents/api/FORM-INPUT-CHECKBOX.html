설명

HTML 체크 박스

사용법

속성별 설명

  • ng-model

    데이터 바인딩할 모델을 적어준다.

  • name - optional

    컨트롤할 컴포넌트의 property 이름을 준다.

  • ng-true-value - optional

    체크 박스가 선택 될 때 true 값을 정의한다. 기본값은 "true".

  • ng-false-value - optional

    체크 박스가 선택이 해제될 때 false 값을 정의한다. 기본값은 "false".

  • ng-change - optional

    체크박스 값이 변경될 때(change 이벤트 발생 시) 해당 {string} 표현식이 실행된다.
