설명

두 멀티 셀렉트의 아이템들을 좌 우로 움직일 수 있게 하는 컴포넌트이다.

사용법

속성별 설명

  • kt-item-selector

    아이템 셀렉터 컴포넌트를 선언하며 $scope에 바인딩된 옵션을 제공할 수 있다. 없으면 공 문자열 입력

    - 옵션 리스트 -

    • showMoveButton : true/false

      아이템을 이동하는 버튼을 보여준다. (기본 : true)

    • showCopyButton : true/false

      아이템을 복사하는 버튼을 보여준다. (기본 : true)

  • left-model

    왼쪽 멀티 셀렉트에서 선택한 모델과 바이딩될 모델명을 적어 준다.

  • right-model

    오른쪽 멀티 셀렉트에서 선택한 모델과 바이딩될 모델명을 적어 준다.

  • left-items

    왼쪽 멀티 셀렉트를 채워줄 $scope에 바인딩된 배열을 적어 준다.

  • right-items

    오른쪽 멀티 셀렉트를 채워줄 $scope에 바인딩된 배열을 적어 준다.

  • left-options

    왼쪽 멀티 셀렉트의 셀렉트 옵션을 적어 준다.

  • right-options

    오른쪽 멀티 셀렉트의 셀렉트 옵션을 적어 준다.
